# 目前的博客使用的Hexo搭建

Hexo是一个快速、简洁且高效的博客框架，基于Node.js构建，支持Markdown语法

# Hexo搭建过程中需注意的问题

编写博客章节的markdown文档一定不要用markdown编辑器,最好用vsCode或者文本编辑器,不然Hexo无法生成.
